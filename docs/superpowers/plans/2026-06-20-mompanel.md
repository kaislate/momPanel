# momPanel Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a graphics-first, frameless desktop info panel (Tauri) for non-technical users that shows clock, date, memory, storage, Wi-Fi, internet, printers, volume, and weather, with safe "open settings" shortcuts, auto-update, and autostart.

**Architecture:** Tauri app = Rust backend + web frontend. The frontend owns all visuals (tile grid, gauges, animations) and is OS-agnostic. The backend is a set of small collector modules, each returning a uniform JSON data shape via Tauri commands/events. Linux is the real implementation; Windows is best-effort for testing with graceful fallbacks.

**Tech Stack:** Tauri 2.x, Rust (`sysinfo`, `serde`, `reqwest`), vanilla HTML/CSS/JS frontend, Tauri plugins: `updater`, `autostart`, `store` (config persistence). Linux native tools: `nmcli`, `lpstat`/CUPS, `wpctl`, `gnome-control-center`. Weather: Zippopotam.us (ZIP→coords) + Open-Meteo (forecast).

## Global Constraints

- **Primary target Linux/Zorin OS (GNOME)**; Windows 11 is test-only with graceful fallbacks — never block startup on a missing platform tool.
- **No direct system mutation** — display + "open native settings screen" only.
- **No API keys** anywhere (Open-Meteo and Zippopotam.us are keyless) — the app installs on other people's machines.
- **Low footprint:** clock/date are frontend-only (no backend call); in-process `sysinfo` reads may run on fast timers; anything that spawns a subprocess runs on a slow timer or is event-driven; polling pauses when the window is hidden/locked.
- **Every collector failure is isolated** to its tile, which shows a calm "not available" state.
- **Uniform data shapes:** the frontend never sees OS branching — only the inside of a collector differs by OS.
- **Auto-update from GitHub Releases**, signed; **Linux ships as AppImage** (required for self-update).
- **Frameless, non-resizable, non-movable** single window.
- Tauri commands return `Result<T, String>` and serialize `T` with `serde`; the frontend treats an error as "not available".

```rust
use std::path::PathBuf;

fn default_clock() -> String { "digital".into() }

#[derive(Serialize, Deserialize)]
pub struct AppConfig {
    #[serde(default)] pub zip: Option<String>,
    #[serde(default = "default_clock")] pub clock_mode: String,
}
impl Default for AppConfig {
    fn default() -> Self { Self { zip: None, clock_mode: default_clock() } }
}
fn path() -> PathBuf {
    let mut p = dirs::config_dir().unwrap_or_else(|| PathBuf::from("."));
    p.push("momPanel"); std::fs::create_dir_all(&p).ok(); p.push("config.json"); p
}
pub fn load() -> AppConfig {
    std::fs::read_to_string(path()).ok()
        .and_then(|s| serde_json::from_str(&s).ok())
        .unwrap_or_default()
}
pub fn save(cfg: &AppConfig) -> Result<(), String> {
    let s = serde_json::to_string_pretty(cfg).map_err(|e| e.to_string())?;
    std::fs::write(path(), s).map_err(|e| e.to_string())
}
```

```js
const tiles = [];
export function registerTile(t) { tiles.push(t); }
export function mountTiles() {
  const grid = document.getElementById('grid');
  tiles.forEach((t, i) => {
    const el = document.createElement('section');
    el.className = 'tile tile--enter';
    el.style.animationDelay = `${i * 70}ms`; // staggered reveal
    el.dataset.id = t.id; grid.appendChild(el); t._el = el;
  });
}
export function startRenderLoop(schedule, fetcher) {
  for (const t of tiles) {
    const ms = schedule[t.id] ?? 0;
    const tick = async () => { t.render(t._el, await fetcher(t.id)); };
    tick();
    if (ms > 0) setInterval(() => { if (!document.hidden) tick(); }, ms);
  }
}
```

```rust
use crate::collectors::types::MemoryData;
use sysinfo::System;

pub fn percent(used_kb: u64, total_kb: u64) -> f32 {
    if total_kb == 0 { return 0.0; }
    (used_kb as f32 / total_kb as f32) * 100.0
}
pub fn read() -> MemoryData {
    let mut s = System::new();
    s.refresh_memory();
    let total = s.total_memory(); let used = s.used_memory(); // bytes in sysinfo 0.30+
    if total == 0 { return MemoryData::unavailable(); }
    MemoryData::ok(percent(used, total), total/1_048_576, used/1_048_576)
}
```
